User Story

As a developer, I can view a plan for collecting the data necessary for the instantiation of the user class and the resource class.

Plan

We will create two front end input fields. One will allow users to create an account using a form, the other will allow logged in users to submit a resource using a separate form. These forms will be used to instantiate a "User" object which will be stored in the User table and "Resource" object which will be stored in the Resource table.
