User Story

As a developer, I can view a plan for collecting the data necessary for the population of the user table and the resource table.

Plan

We will create two front end input fields. One will allow users to create an account using a form, the other will allow logged in users to submit a resource using a separate form. These forms will be used to instantiate an "User" object which will be stored in the User table and a "Resource" object which will be stored in the Resource table.
